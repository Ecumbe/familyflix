📋 Cómo configurar el Worker (una sola vez)

1
Ve a dash.cloudflare.com → Workers & Pages → Create → Hello World
2
Ponle el nombre familyflix-worker, haz clic en Deploy, luego Edit code y pega el contenido del archivo familyflix-worker.js que descargaste. Guarda y despliega.
3
En el Worker ve a Settings → Variables and Secrets y agrega estas tres variables:
GEMINI_KEY = tu Gemini API Key
ADMIN_SECRET = una contraseña que tú elijas (la misma de arriba)
4
En el Worker ve a Settings → Bindings → Add → R2 Bucket
Variable name: R2_BUCKET · Bucket: familyflix
5
Copia la URL del Worker (algo como familyflix-worker.TU-USUARIO.workers.dev) y pégala arriba.

📋 Cómo configurar Google Apps Script (una sola vez)

1
Ve a script.google.com → Nuevo proyecto
2
Borra el código que viene por defecto y pega el contenido del archivo familyflix-appscript.js. Guarda con Ctrl+S.
3
Haz clic en Implementar → Nueva implementación → tipo: Aplicación web → Acceso: Cualquier persona → Implementar
4
Copia la URL que aparece y pégala arriba en Apps Script URL.
